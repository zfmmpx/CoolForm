# umi project

## Getting Started

推荐使用 yarn 管理 npm 依赖，并使用国内源（阿里用户使用内网源）。

```
# 国内源
$ npm i yarn tyarn -g
# 后面文档里的 yarn 换成 tyarn
$ tyarn -v

# 阿里内网源
$ tnpm i yarn @ali/yarn -g
# 后面文档里的 yarn 换成 ayarn
$ ayarn -v
```

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```
